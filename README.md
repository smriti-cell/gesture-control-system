# Gesture Control System

A computer vision-based project that enables touch-free interaction through real-time hand gesture recognition using Python and OpenCV.

## Features
- Real-time hand detection
- Gesture recognition
- Touch-free computer interaction

## Technologies Used
- Python
- OpenCV
- Machine Learning

## Files
- app.py - Main application
- collect_data.py - Dataset collection
- train_model.py - Model training
- gesture_data.csv - Dataset
